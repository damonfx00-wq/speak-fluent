## üîß Database Setup (Optional)

The app will work without a database, but some features require it.

### Option 1: Use SQLite (Easiest)
Edit `backend/.env`:
```env
DATABASE_URL=sqlite:///./speak_fluent.db
```

### Option 2: Use PostgreSQL
1. Install PostgreSQL
2. Create database:
   ```sql
   CREATE DATABASE speak_fluent;
   CREATE USER your_user WITH PASSWORD 'your_password';
   GRANT ALL PRIVILEGES ON DATABASE speak_fluent TO your_user;
   ```

3. Update `backend/.env`:
   ```env
   DATABASE_URL=postgresql://your_user:your_password@localhost/speak_fluent
   ```

### Option 3: Skip Database
The app works without a database! Dynamic content is generated by AI.
